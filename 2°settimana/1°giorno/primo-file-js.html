Oggetto console

La console è uno strumento fondamentale per scrivere codice javascript (per qualsiasi linguaggio di programmazione)

La console ci permette di controllare il flusso del nostro codice

Spesso la console si usa proprio per tenere traccia dell'evoluzione del mio script o verificare dei valori

per vedere la console tasto destro-> ispeziona-> console

Var, let, const

var è la keyword usata da javascript per usare una variabile, var è accessibile in qualunque parte dello script

let introdotta da Ecmascript6 è accessibile in un determinato amibito (scope) che è delimitato da un blocco di codice

Un blocco di codice javascript è una serie di istruzioni delimitate da { .. blocco di codice}

const

Introdotte da Ecmascript6-> istruisce l'engine che il valore assegnato non è modificabile
