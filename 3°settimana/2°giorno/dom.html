Dom

Document obcject model è in realtà dell'oggetto window e rappresenta l'intera struttura html caricata nella mia pagina, ha una struttra a oggetto. Il DOM fornisce una rappresentazione gerarchica di oggetti, che viene spesso chiamata Dom tree (assomiglia a un albero genealogico). Questo albero di oggetti è accessibile tramite javascript, infatti possiamo fare

  • leggere gli elementi del DOM
  • modificare la struttura degli elementi (tag)
  • modificare il contenuto degli elementi (tag)
  • modificare lo stile

Il DOM si presenta come un albero con una serie di nodi che hanno diversi tipi

Nelle specifiche W3C esistono altri tipi di nodi: ad esempio commento, document type, ma li trascuriamo perchè non faremo mai javascript su questi titpi di nodo

Manipolare il DOM via javascript

  1. Selezionare un elemento del DOM
  2. Modificare lo "style" di un elemento
  3. Modificare le classi di un lelemento
  4. Modificare il contenuto di un elemento
  5. Creare, inserire e rimuovere degli elementi nel DOM
